Why journalot?

Every journaling app has the same problem: friction. Open app → wait for sync → click new entry → choose template...

journalot eliminates all of that:

Already in Terminal

You're already there. Just type journal and you're writing. Zero context switching.

Your Editor, Your Rules

Opens in your preferred editor (vim, nano, VSCode). No learning curve, no compromises.

Git-Native Versioning

Auto-commits to Git. Full history, sync across devices, never lose an entry.

Just Markdown Files

Complete ownership. Your data, your format, forever accessible. No vendor lock-in.

Quick Capture

journal "text" appends instantly. Perfect for fleeting thoughts and rapid logging.

Powerful Search

Find anything with --search, filter by --tag, view stats, export to HTML/PDF.

Features

Core Features (Free Forever)

  • Daily journal entries in markdown format
  • Quick capture mode for instant thoughts
  • Search entries by keyword or tag
  • Week view - open all entries from current week
  • Stats dashboard (entries, words, patterns)
  • Export to HTML and PDF
  • Template support for customized entries
  • Daily writing prompts
  • Random entry rediscovery
  • Git diff view for entry history
  • Archive old entries by year

Pro Features

  • AI-powered insights and pattern analysis
  • Voice capture with transcription
  • GPG encryption for sensitive entries
  • Cloud backup to S3/Dropbox
  • Graph visualization of tags and topics
